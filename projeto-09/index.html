Tipos de Variaveis

Variáveis são como uma memória de computador. Elas guardam diversos tipos de informações. JavaScript tem três tipos:

  • Var: Antigamente, qualquer tipo de variável em JavaScript se chamava var. Então para se definir uma variável, você escreveria 
    var nomeVariavel = 10;
  • Const: Para armazenar valores constantes, que não mudam ao longo do código.
  • Let: Para definir variáveis cujo valor pode mudar a qualquer momento.

O primeiro tipo entrou quase em desuso, sendo mais utilizado em aplicações mais antigas. Para definir uma variável é facil, basta colocar:
const nomeDaVariavel = 'olá, eu sou uma variável'

Em JavaScript, podemos guardar quase qualquer coisa dentro de uma variável. Pode ser um número, uma string, um valor booleano, uma função, um objeto, entre várias outras coisas.

Funções

Funções são blocos de código que podemos envocar sempre que precisarmos, elimanando assim a necessidade de refazer uma mesma estrutura. Para definir uma variável fazemos:

                        
                            function functionName (parametro01, parametro02) {
                                console.log('Você pode colocar o que quiser dentro de uma variável, até mesmo definir um objeto.')
                                return parametro01 + parametro02 + 3;
                            }
                            functionName(1,2);

Os parametros de uma função podem ser qualquer coisa. Uma string, um número. O return retorna o valor, qualquer valor ou calculo que você colocar depois dele, ao invés de retornar a função. Assim você pode armazenar o resultado daquilo que a função fez para você.

Loops

Loops são estruturas de repetição. Elas se repetem até que um determinado limite seja atingido. São algumas delas: for e switch.

                        
                            for (i = 0; i > 3; i++) {
                                console.log('Esta frase vai ser escrita na tela até que i seja maior que três, ou seja, igual a três');
                            }
                            switch (variavel) {
                                case valor:
                                    console.log("Se 'valor for igual ao que tá na variavel, escreva esse console.")
                                break;
                                case valor2;
                                    console.log("Se o valor2 for igual ao valor da variável, escreve este console.")
                                break;
                                default:
                                    console.log("Se nenhuma das opções for igual ao valor da variavel, escreva esta mensagem no console.")
                            }

Então, o Switch, faz uma repetição até que o valor presente na condição seja igual a algum valor dos cases

Objetos

Na vida real, um carro é um objeto. Uma armário é um objeto. Uma colher é um objeto. Todos esses objetos tem suas propriedades. Uma propriedade/característica do carro: ser azul. Outra propriedade: ser da honda. Honda e azul são duas propriedades possíveis de um carro. Em javaScript, escreveriamos as propriedades de um carro assim: 

                            
                                const carro = {marca: 'honda', cor: 'azul'};
                            
                            Onde:
                            Carro é um objeto;
                            marca é uma propriedade que todos os carros têm;
                            Honda é uma das possíveis marcas do objeto. Em javaScript, chamamos isso de valor.

Só que em JavaScript os objetos não são carros, nem armarios ou garfos. Geralmente, um objeto é um conjunto de código ou parte de um arquivo html. E esse conjunto de informações é guardado dentro de um objeto, que por sua vez é guardado dentro de uma variável. Pense em objeto como um armazem de informações, que podem ser acessadas e acordo com sua área (categoria). Isso é um objeto.
